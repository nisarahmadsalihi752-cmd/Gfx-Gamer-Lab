📢 ADS MANAGEMENT

Adsterra Se Ad Code Kaise Lagayein:

1. Adsterra.com pe login karein
2. Publishers → Sites → Add Site karein
3. Site verify hone ke baad Ad Units → Create New
4. Type chunein: Display Banner (300x250 ya 728x90)
5. Jo HTML/Script code mile woh neeche textbox mein paste karein
6. Show toggle ON karein aur Save karein

Smartlink: Adsterra → Publishers → Smartlinks se link copy karein

🔗 SMARTLINK (Download Buttons — 1st Click Ad)

⭐ Jab user kisi DOWNLOAD button pe pehli baar click kare to yeh link khulta hai

🔗 EXTERNAL LINK AD (Extra Buttons — 1st Click Ad)

⭐ Jab user Extra Button (Apply Process, Facebook, etc.) pe pehli baar click kare
